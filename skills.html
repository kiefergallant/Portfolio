    • HTML5

    In the Web Developer program at SAIT we learned Hypertext Markup Language. Some of the tags we covered in the HTML portion of the course were paragraph , heading ,horizontal rule,strong and em , line break , anchor , divs and spans , wrappers, ordered and unordered lists, header, nav , section tags.We also learned about the common naming conventions for file names, tables and forms.We also learned about simple but eye catching animations using transform translate, transition delay and transition.

    CSS3

    We have explored concepts such as CSS selectors and rules, internal stylesheets , external stylesheets , inline stylesheets(bad practice),element selectors, ID selectors , class selectors,units of measurement,font styles,color styles,box model, inline block, float method, media queries, float grid and flexbox.




    SASS

    Sass is a language that adds to CSS that's used to describe the style of a document cleanly and structurally, with more power than flat CSS allows. Sass both provides a simpler, more elegant syntax for CSS and implements various features that are useful for creating manageable stylesheets.




    Bootstrap

    Bootstrap is a framework to help you design websites faster and easier. It includes HTML and CSS based design templates for typography, forms, buttons, tables, navigation, modals, image carousels, etc. It also gives you support for JavaScript plugins.




    JavaScript

    In our JavaScript portion of the course we learned about numbers , strings , objects , arrays and functions. We also learned about changing HTML content with JavaScript methods such as getElementById. We also learned about using dates and times in JavaScript.




    MySQL

    We learned MySQL syntax and how to use the GUI for SQL. We also learned how to combine PHP and SQL to make databases of usernames and passwords and information such as first , last name , the time stamp when they created their account and proper storage of passwords.




    Python

    Python is my favorite language because of its simple syntax and easy to remember commands and methods. The first language I taught myself was python because I was interested in Data Science and the analyisis of large datasets. Python can do many things because it has so many libararies




    PHP

    PHP is the backend language we learned in the SAIT FastTrack Web Developer program. We learned how to integrate databases into our websites with PHP and MySQL.PHP is a general use scripting language and a server side language. We also learned how to run a local server on our computers.




    GitHub

    We learned that GitHub is a version control system that tracks the changes made to a project. Some of the specific skills we learned were how to create and use a repository, how to start a branch, how to make changes to a file and then commit then and then push them to the repo. We also learned about merging branches to the master. Another thing we leared was how to assign issues.
